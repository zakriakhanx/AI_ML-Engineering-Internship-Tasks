# ML Practice Projects

A collection of machine learning projects covering classification, regression, and data visualization tasks.

## Projects

| # | Project | Description |
|---|---------|-------------|
| 1 | [Exploring_and_Visualizing_a_Simple_Dataset](./Exploring_and_Visualizing_a_Simple_Dataset/README.md) | EDA and visualization using the Iris dataset |
| 2 | [General_Health_Query_Chatbot](./General_Health_Query_Chatbot/README.md) | AI chatbot with safety guardrails for health queries |
| 3 | [Heart_Disease_Prediction](./Heart_Disease_Prediction/README.md) | Heart disease prediction using Logistic Regression |
| 4 | [House_Price_Prediction](./House_Price_Prediction/README.md) | House price prediction using Linear Regression |
| 5 | [Predict_Future_Stock_Prices](./Predict_Future_Stock_Prices/README.md) | Stock price prediction using Linear Regression and Random Forest |

## Quick Start

Each project has its own directory with:
- A `README.md` with detailed documentation
- A main Python script (e.g., `predict.py`, `chatbot.py`)
- Dataset files (`.csv`)
- Original task description (`task.md`)

To run any project:
1. Navigate to the project directory
2. Install dependencies (see project's README)
3. Run the main script
